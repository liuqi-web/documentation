    # Attributes

    参数 说明 类型 可选值 默认值
    text 显示的文本内容 String - -
    row 超出几行后隐藏 Number - 7
    openText 超出后展开按钮文本 String - 展开
    closeText 超出后隐藏按钮文本 String - 收起
